## 1. Vertex data

Each vertex stores position (vec4), color (vec4), and UV (vec2). Pack them into a `Float32Array` and upload once:

```tsx
const verticesBuffer = device.createBuffer({
  size: cubeVertexArray.byteLength,
  usage: GPUBufferUsage.VERTEX,
  mappedAtCreation: true,
});
new Float32Array(verticesBuffer.getMappedRange()).set(cubeVertexArray);
verticesBuffer.unmap();
```

Describe the layout in the pipeline so the vertex shader receives `@location(0)` position and `@location(1)` uv:

```tsx
vertex: {
  module: device.createShaderModule({ code: basicVertWGSL }),
  buffers: [{
    arrayStride: cubeVertexSize,
    attributes: [
      { shaderLocation: 0, offset: cubePositionOffset, format: "float32x4" },
      { shaderLocation: 1, offset: cubeUVOffset, format: "float32x2" },
    ],
  }],
},
```

## 2. Depth testing

Without a depth attachment, back faces draw on top of front faces. Add `depth24plus` from [Depth & MSAA](/docs/rendering/depth-and-msaa):

```tsx
depthStencil: {
  depthWriteEnabled: true,
  depthCompare: "less",
  format: "depth24plus",
},
primitive: { topology: "triangle-list", cullMode: "back" },
```

Create a matching depth texture sized to the canvas and clear it to `1.0` each frame. Recreate the depth texture when the canvas resizes.

## 3. MVP uniform

The vertex shader multiplies positions by a single `mat4x4f` uniform:

```wgsl
struct Uniforms {
  modelViewProjectionMatrix: mat4x4f,
}
@group(0) @binding(0) var<uniform> uniforms: Uniforms;

@vertex
fn main(@location(0) position: vec4f, @location(1) uv: vec2f) -> VertexOutput {
  output.Position = uniforms.modelViewProjectionMatrix * position;
  output.fragPosition = 0.5 * (position + vec4(1.0));
}
```

Build the matrix on the CPU each frame with `wgpu-matrix`:

```tsx
const projection = mat4.perspective((2 * Math.PI) / 5, aspect, 1, 100);
const view = mat4.identity();
mat4.translate(view, vec3.fromValues(0, 0, -4), view);
mat4.rotate(view, vec3.fromValues(Math.sin(t), Math.cos(t), 0), 1, view);
const mvp = mat4.multiply(projection, view, mat4.create());

device.queue.writeBuffer(
  uniformBuffer, 0,
  mvp.buffer, mvp.byteOffset, mvp.byteLength,
);
```

<Callout type="info" title="multiply order">
`mat4.multiply(a, b, out)` writes `out = a × b`. Pass a fresh `mat4.create()` as the third argument - it is the **output**, not a third matrix to multiply.
</Callout>

## 4. Animation loop

Wrap setup in `requestAnimationFrame`. Each tick: update uniforms, encode the pass, submit, **`present()`**:

```tsx
const render = () => {
  updateMvpAndResizeDepth();
  // beginRenderPass → setPipeline → setBindGroup → setVertexBuffer → draw(36)
  device.queue.submit([encoder.finish()]);
  context.present();
  requestAnimationFrame(render);
};
```

A cube has 36 vertices (12 triangles × 3).
