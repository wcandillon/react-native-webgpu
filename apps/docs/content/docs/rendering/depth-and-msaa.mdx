When drawing 3D geometry, pixels compete for the same screen location. **Depth testing** ensures closer triangles occlude farther ones. **MSAA** (multisample anti-aliasing) smooths jagged edges.

[Rotating Cube](/docs/rendering/rotating-cube) uses depth testing - this page explains the concepts in depth.

## Depth buffer

Enable depth in the render pipeline:

```tsx
depthStencil: {
  depthWriteEnabled: true,
  depthCompare: "less",
  format: "depth24plus",
},
primitive: {
  topology: "triangle-list",
  cullMode: "back", // skip back faces
},
```

Create a **depth texture** matching your canvas size:

```tsx
const depthTexture = device.createTexture({
  size: [canvas.width, canvas.height],
  format: "depth24plus",
  usage: GPUTextureUsage.RENDER_ATTACHMENT,
});
```

Attach it in the render pass:

```tsx
depthStencilAttachment: {
  view: depthTexture.createView(),
  depthClearValue: 1.0,
  depthLoadOp: "clear",
  depthStoreOp: "store",
},
```

Clear depth to `1.0` (far plane). Fragments with lower depth values pass the `"less"` test.

**Recreate the depth texture when the canvas resizes** - stale dimensions cause clipping artifacts.

## MSAA

Multisampling renders edges at higher resolution and resolves to the swapchain texture:

```tsx
const msaaTexture = device.createTexture({
  size: [canvas.width, canvas.height],
  sampleCount: 4,
  format: presentationFormat,
  usage: GPUTextureUsage.RENDER_ATTACHMENT,
});

// In render pass:
colorAttachments: [{
  view: msaaTexture.createView(),
  resolveTarget: swapchainTextureView,
  clearValue: [0, 0, 0, 1],
  loadOp: "clear",
  storeOp: "store",
}],
```

Set `sampleCount: 4` on the render pipeline to match the MSAA texture.

MSAA costs GPU memory and fill rate - use it for 3D scenes with visible aliasing, skip for full-screen 2D effects.

## Reversed-Z (advanced)

Standard depth uses `0 = near, 1 = far` with `depthCompare: "less"`. **Reversed-Z** maps near to `1.0` and far to `0.0` with `depthCompare: "greater"` for better floating-point precision in large scenes. Requires consistent projection matrix setup across all passes.
