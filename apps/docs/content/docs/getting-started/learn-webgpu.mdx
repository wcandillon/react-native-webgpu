Because WebGPU is a low-level API, people often use higher-level libraries on top of it, such as Three.js or Tensorflow.js.
That being said, WebGPU is a very elegant API, and the concepts are genuinely fun to learn.
There is a fantastic [WebGPU Fundamentals](https://webgpufundamentals.org/) site that teaches the fundamentals from the ground up.

There is also great [React Native WebGPU tutorial](https://www.youtube.com/watch?v=vons_XypdQw) by Daniel Friyia that takes you through the base concepts of the WebGPU render pipeline.
