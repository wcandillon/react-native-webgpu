---
title: Offscreen Canvas
description: Headless GPU renderings with CPU readbacks
---

`GPUOffscreenCanvas` is a helper shim that mimics the Web [`OffscreenCanvas`](https://developer.mozilla.org/en-US/docs/Web/API/OffscreenCanvas) API. It is offered as a convenience to quickly produce offscreen renderings with CPU readback.
## Basic usage

```tsx twoslash
declare const adapter: GPUAdapter;
declare const encoder: GPUCommandEncoder;
// ---cut---
import { GPUOffscreenCanvas } from "react-native-webgpu";

const canvas = new GPUOffscreenCanvas(256, 256);
const context = canvas.getContext("webgpu")!;

const device = await adapter.requestDevice();
context.configure({ device, format: "rgba8unorm" });

// encode render passes into the offscreen target…
device.queue.submit([encoder.finish()]);

const pixels = await canvas.getImageData(); // CPU readback
```
