The `RNWebGPU` global is installed when you import `react-native-webgpu`. It provides native interop beyond standard WebGPU.

## Properties

| Member | Description |
| ------ | ----------- |
| `gpu` | The `GPU` instance (same as `navigator.gpu`) |
| `fabric` | Whether Fabric renderer is active |
| `getNativeSurface(contextId)` | Resolve native surface for a canvas context id |
| `MakeWebGPUCanvasContext(id, w, h)` | Create RN canvas context (web polyfill) |
| `DecodeToUTF8(buffer)` | Decode bytes to UTF-8 string |
| `createImageBitmap(buffer)` | Decode image bytes to `ImageBitmap` |
| `loadVideoFrame(path)` | Load a video frame from file path |
| `createTestVideoFrame(w, h)` | Create synthetic test frame |
| `createVideoFrameFromNativeBuffer(pointer)` | Wrap native buffer (e.g. Vision Camera) |
| `createVideoPlayer(path, pixelFormat?)` | Create a decoded video player |
| `writeTestVideoFile()` | Write a test video file, return path |

You rarely call `MakeWebGPUCanvasContext` directly — [`CanvasRef.getContext("webgpu")`](/api/canvas-ref) does this for you.

## Usage

### Video player

Stream decoded MP4 frames into WebGPU:

```tsx
import { RNWebGPU } from "react-native-webgpu";

const player = RNWebGPU.createVideoPlayer(
  "https://example.com/video.mp4",
  "nv12", // or "bgra8" for shared texture memory import
);

player.play();

const render = () => {
  const frame = player.copyLatestFrame();
  if (frame) {
    const texture = device.importExternalTexture({ source: frame });
    // draw with texture_external sampler in WGSL…
    frame.release(); // after queue.submit() that sampled it
  }
  requestAnimationFrame(render);
};

// When done:
player.release();
```

See [Video Player](/docs/native-media/video-player) for the full render loop with `present()`.

### Image decode

Upload encoded image bytes without a separate decoder:

```tsx
const response = await fetch("https://example.com/texture.png");
const bytes = await response.arrayBuffer();
const bitmap = await RNWebGPU.createImageBitmap(bytes);

device.queue.copyExternalImageToTexture(
  { source: bitmap },
  { texture: gpuTexture },
  { width: bitmap.width, height: bitmap.height },
);
bitmap.close();
```

### Vision Camera frame processor

Wrap a native buffer pointer from a frame processor:

```tsx
import { RNWebGPU } from "react-native-webgpu";

const processFrame = (frame: Frame) => {
  "worklet";
  const nativeFrame = RNWebGPU.createVideoFrameFromNativeBuffer(
    frame.nativeBuffer.pointer,
  );

  const texture = device.importExternalTexture({ source: nativeFrame });
  // render…
  nativeFrame.release();
};
```

See [Vision Camera](/docs/native-media/vision-camera) for device setup on the UI thread.

### Resolve native surface by context id

```tsx
const contextId = canvasRef.current!.getContextId();
const surface = RNWebGPU.getNativeSurface(contextId);
// surface: { surface: bigint, width, height, clientWidth, clientHeight }
```

### Test helpers

```tsx
// Synthetic frame for unit tests (no real camera/video file)
const frame = RNWebGPU.createTestVideoFrame(640, 480);

// Write a short test MP4 to disk and get the path
const testPath = RNWebGPU.writeTestVideoFile();
const player = RNWebGPU.createVideoPlayer(testPath, "nv12");
```

## Pixel formats

`NativeVideoPixelFormat`: `"bgra8"` | `"nv12"`

| Format | Import path | WGSL sampler |
| ------ | ----------- | ------------ |
| `"nv12"` | `importExternalTexture` | `texture_external` + `textureSampleBaseClampToEdge` |
| `"bgra8"` | `importSharedTextureMemory` | regular `texture_2d` |
